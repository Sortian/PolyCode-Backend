# PolyCode JavaScript Certificate — Requirements

Earn the **PolyCode JavaScript Developer Certificate** by completing this structured course and demonstrating practical skills.

## Who this certificate is for

- Students who finished all five learning modules
- Self-taught developers who want a verified learning path
- Anyone preparing for junior JavaScript or full-stack roles

## Requirements to earn your certificate

### 1. Complete all modules (required)

Study every lesson in order:

- **Module 01 — Beginner** (8 lessons)
- **Module 02 — Intermediate** (5 lessons)
- **Module 03 — Advanced** (7 lessons)
- **Module 04 — Professional** (7 lessons)
- **Module 05 — Mastery** (4 lessons)

You should understand each topic well enough to explain it in your own words.

### 2. Complete hands-on projects (required)

Submit or complete **at least 3 projects**:

| Project | Module | Minimum scope |
|---------|--------|---------------|
| Console or browser app | 01 Beginner | Variables, functions, arrays/objects |
| Interactive web page | 02 Intermediate | DOM + fetch API or events |
| API or full-stack app | 04 Professional | Node/Express or React basics |

The **E-commerce capstone** in Module 05 counts as one professional-level project.

### 3. Pass the final assessment (required)

- Read [Final assessment](02-final-assessment.md)
- Score **70% or higher** on all sections combined
- Complete the practical coding challenge without copying full solutions

### 4. Code quality standards (required)

Your submitted work should:

- Use `const` / `let` (avoid `var` in new code)
- Use meaningful variable and function names
- Handle errors with `try/catch` where appropriate
- Include brief comments for non-obvious logic

## Skills you will have earned

After certification, you should be able to:

- Write clean JavaScript from beginner to intermediate level
- Manipulate the DOM and handle async operations
- Explain closures, prototypes, and the event loop
- Build a simple REST API with Express
- Write basic tests and follow security best practices
- Solve common algorithm problems with arrays and objects

## Recommended study schedule

| Week | Focus |
|------|-------|
| 1–2 | Module 01 + first mini project |
| 3–4 | Module 02 + interactive page |
| 5–6 | Module 03 |
| 7–9 | Module 04 + API project |
| 10–12 | Module 05 + capstone + exam |

## After you pass

1. Review any weak topics from the assessment
2. Add capstone projects to your portfolio
3. Continue with TypeScript, advanced React, or backend depth

Return to the [Course roadmap](00-roadmap.md) anytime to track your progress.
