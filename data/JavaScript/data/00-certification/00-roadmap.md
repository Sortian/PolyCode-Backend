# JavaScript Certificate Course — Learning Roadmap

Welcome to the **PolyCode JavaScript Certificate Course**. This is a structured path from complete beginner to job-ready developer. Follow the modules in order — each lesson builds on the previous one.

## How this course works

| Element | What you do |
|--------|-------------|
| **Modules 01–05** | Read lessons in order, run code examples, complete exercises |
| **Practice** | Build small programs after each module before moving on |
| **Assessment** | Pass the final exam in Module 00 after finishing Module 05 |
| **Certificate** | Complete all modules + pass assessment → earn your PolyCode JS certificate |

**Estimated time:** 8–12 weeks at 5–8 hours per week.
