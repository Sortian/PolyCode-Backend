## 🎯 Level 1: Console Calculator

**Duration:** 1 hour | **Difficulty:** Beginner

### What You'll Build

A command-line calculator that performs basic math operations.

### Concepts Used

- Variables and basic data types
- User input and output
- Methods and parameters
- Conditionals and loops
- Error handling basics

### Project Structure

```
ConsoleCalculator/
├── Program.cs           # Main entry point
├── Calculator.cs        # Calculator logic
└── README.md           # Instructions
```

### Learning Outcomes

- [ ] Take user input with Console.ReadLine()
- [ ] Parse strings to numbers
- [ ] Perform calculations
- [ ] Display results
- [ ] Handle invalid input

### Get Started

👉 Open [01-Console-Calculator/README.md](01-Console-Calculator/README.md)

## 🎓 How to Use These Projects

### **For Each Project:**

1. **Read the README** - Understand requirements
2. **Study the starter code** - See the structure
3. **Implement step-by-step** - Don't try to do everything at once
4. **Test frequently** - Run code often
5. **Check the solution** - Only after you try!
6. **Modify and extend** - Make it your own

### **Tips:**

- ✅ Don't copy-paste - Type everything
- ✅ Experiment - Change things and see what breaks
- ✅ Ask "Why?" - Understand decisions
- ✅ Break it on purpose - Learn recovery
- ✅ Add features - Go beyond requirements
- ✅ Show someone - Get feedback
