# Level 2: Object-Oriented Programming (OOP) ⭐⭐

**Duration:** 2-3 weeks | **Prerequisites:** Level 1 | **Difficulty:** Intermediate

## 🎯 What You'll Learn

Master object-oriented programming - the core of C# and modern development:

- ✅ Classes and objects (blueprints and instances)
- ✅ Properties, fields, and methods
- ✅ Constructors and initialization
- ✅ Inheritance (code reuse)
- ✅ Polymorphism (same interface, different behavior)
- ✅ Interfaces and abstract classes
- ✅ Exception handling (try/catch/finally)
- ✅ Collections and data organization

## 🎯 Learning Objectives Checklist

By the end of Level 2, you should be able to:

- [ ] Design and build classes to model real-world objects
- [ ] Create and use object instances

- [ ] Use interfaces to define contracts

- [ ] Recognize when to use inheritance vs interfaces

## 💡 OOP Design Principles (Preview)

You'll learn these in Level 4, but keep them in mind:

- **SOLID Principles** - Write maintainable code
- **DRY (Don't Repeat Yourself)** - Avoid duplication
- **KISS (Keep It Simple)** - Don't over-engineer
- **YAGNI (You Aren't Gonna Need It)** - Don't add features you don't need
