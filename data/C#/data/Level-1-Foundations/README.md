## 💡 Common Beginner Mistakes

1. **Off-by-one errors** - Remember arrays are 0-indexed!
2. **Forgetting braces** - `if(x > 5) y = 10;` vs `if(x > 5) { y = 10; z = 20; }`
3. **String vs int** - `"5" + "10"` = `"510"` not `15`
4. **Single = vs double ==** - Assignment vs comparison
5. **Division by zero** - Always check!

See [Common Mistakes](../Reference/common-mistakes.md) for more.
